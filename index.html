Personal projects

Hangman Discord Bot

Hangman is a Discord bot that allows playing Hangman on Discord. Its primary purpose is to help people learn English.

The bot is available in probably the biggest Discord community for people who want to learn languages (over 5k members). The community is a reminiscent of the old language websites SharedTalk and HelloLingo.

The bot is integrated with the Collins Dictionary API.

Stack:

  • Deno (hosted at Heroku inside a Docker container)
  • TypeScript
  • MongoDB (Atlas cloud)
  • Redis (redis.com cloud)

Discord server: https://discord.gg/NSEF7JWEgz.

Repository: https://github.com/Leokuma/hangman.
